# DjangoToDoProject

Простое и удобное веб-приложение для управления списком задач (To-Do List), созданное на фреймворке Django.

## ✨ Основные функции
Проект позволяет полностью контролировать свой рабочий процесс:
* **Создание задач:** добавление новых дел с указанием названия и подробного описания.
* **Управление статусом:** возможность помечать задачи как выполненные или возвращать их в работу.
* **Редактирование:** изменение названия или описания уже существующих задач.
* **Удаление:** очистка списка от ненужных или завершенных записей.
* **Детализация:** каждая задача содержит заголовок, текстовое описание и индикатор выполнения.

## 🛠 Стек технологий
* **Python 3**
* **Django** (Web Framework)
* **SQLite** (База данных)
* **HTML/CSS** (Интерфейс пользователя)

## 📦 Инструкция по запуску

1. **Клонируйте проект:**
   ```bash
   git clone [https://github.com/ваш-логин/DjangoToDoProject.git](https://github.com/ваш-логин/DjangoToDoProject.git)
   cd DjangoToDoProject
Настройте окружение и установите зависимости:

Bash
python3 -m venv .venv
# Если вы на Linux (например, Mint):
./.venv/bin/python3 -m pip install django
Примените миграции:

Bash
./.venv/bin/python3 manage.py makemigrations
./.venv/bin/python3 manage.py migrate
Запустите сервер:

Bash
./.venv/bin/python3 manage.py runserver
После запуска проект будет доступен по адресу: http://127.0.0.1:8000/
